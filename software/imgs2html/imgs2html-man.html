NAME

imgs2html - generate HTML files from images files

SYNOPSIS

imgs2html [--suffix=SUFFIX] [--thumbnail-ratio=SIZE] [--homepage-url=URL] [--icon-dir=DIR] [--style=URL] [--icon-role-file=FILE] [--nav-role-str=STR] [--logo] [--notitle] [--version] [--help]

DESCRIPTION

imgs2htmlは複数の画像ファイルを元にHTMLファイルを作成します。

このスクリプトを実行するとカレントディレクトリにある画像ファイルを探し、 それらの一覧と各画像からなるHTMLファイル群を生成します。

生成されるHTMLファイルの文字コードはISO-2022-JPです。

OPTIONS

以下のオプションを指定できます。

-s, --suffix=SUFFIX

対象とする画像の拡張子を指定します。 （デフォルトは gif です）

-r, --thumbnail-ratio=SIZE

サムネール画像の元画像からの比率を指定します。 （デフォルトは 15% です）

-u, --homepage-url=URL

ホームのURLを指定します。 （デフォルトは ../ を指します。）

--icon-dir=DIR

ナビゲーション・アイコンのディレクトリ （デフォルトは ../image/ です）

--style=URL

HTMLファイルで利用するスタイルシートのURLを指定します。 （デフォルトではスタイルシートは使いません）

--logo

ロゴを表示します。 （デフォルトでは表示しません）

--notitle

タイトル（○枚目）を表示しません。 （デフォルトでは表示します）

(ナビゲーションバーの設定)

ナビゲーションバーの部分のアイコンや文字列などを変更するには、以下のオ プションが利用できます。

roleの部分にはそれぞれ、 first, prev, next, last, logo, index, home のいずれかが入ります。

--icon-role-file=FILE

ナビゲーションバーの各位置のアイコンのファイル名を指定します。 （ファイルが存在しない場合はアイコンは表示しません）

--nav-role-str=STR

ナビゲーションバーの各位置の説明文字列を設定します。

デフォルトの設定は以下の通り：

--icon-first-file first.gif
--icon-prev-file prev.gif
--icon-next-file next.gif
--icon-last-file last.gif
--icon-home-file home.gif
--icon-index-file info.gif
--icon-logo-file logo.gif
--nav-first-str 最初のページ
--nav-prev-str 前のページ
--nav-next-str 次のページ
--nav-last-str 最後のページ
--nav-home-str ホームページ
--nav-index-str 一覧
--nav-logo-str ロゴ
--version

バージョンを表示します。

-h, --help

このヘルプを表示します。

PREREQUISITES

このスクリプトは、画像の処理にImageMagickを利用します。 また、日本語のコード変換にnkfを利用します。

COREQUISITES

ps2imgsスクリプトと一緒に使うと効果的かもしれませんね。
