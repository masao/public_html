imgs2html & ps2imgs

PostScriptファイルをWeb用の画像形式に変換するスクリプト（ps2imgs）、 たくさんの画像ファイルを一気にHTMLファイルへ変換するスクリプト（imgs2html）の2つのPerlスクリプトを置いています。

例えば、こちらのような複数の画像とその一覧からなるページを作れます。

バグ報告・要望・感想などは masao@ulis.ac.jp までお寄せ下さい。お気軽にどうぞ。

類似ソフトウェア

（他にもご存じの方はお知らせ下さい。）

  • WebMagick
  • thumbnail_index
  • MagicPoint: プレゼンテーションツール。
  • gPhoto: デジタルカメラ用ソフトウェア。サムネール画像を含むHTMLを出力する機能がある。
  • SWIGS: Structured Web Image Gallery system
  • RubyPhoto: SWIGSを参考にして作られたもの。XMLでのテンプレート記述。
  • BINS: SWIGSの拡張版。gettextを使った国際化など。
  • mkthum.pl: デジカメ画像の整理用!?（Exif対応）
  • cthumb
  • Gallery.py
  • Gallery
  • IDS
  • Gimp Gallery: HTMLを生成するGIMP。
