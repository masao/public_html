ファイル名一括置換の方法

filename.txtをfilename.bakに拡張子を変換する方法です。きちんと確認はしていません。

csh版

foreach file (*.txt); mv $file $file:r.bak; end;

sh版

for file in *.txt; do mv $file `basename $file .txt`.bak; done;

sed版

ls | sed -e 's/\(\(.\)\.txt\)$/mv \1 \2.bak/' | sh
