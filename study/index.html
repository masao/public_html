関連サイト

VRML

  • VRML Repository in Japan by Y.Andoh
    日本語によるVRML情報へのリンク集。
    VRML FAQ日本語版、仕様書の日本語訳(VRML1.0,VRML2.0)、VRMLメーリングリストのアーカイブなど、貴重な日本語によるVRML情報がいっぱい。
  • VRML Repositry(EDSC)
    VRMLに関する膨大なリンク集。
    情報がよく整理され非常に有用。検索も可。明確なポインタがなければ、まずここへ。
  • SGI社のVRMLサイト
    業界最大手シリコングラフィック社のページ。VRMLのいろいろなコンテンツが見れる。
    VRMLブラウザCosmo Player、EAI(External Application Interface)など。
  • VRMLコンソーシアム
    VRMLの仕様などの決定などを行なっている公式機関。仕様書など、ドキュメント多数。
  • VRML Forum
    VRMLの仕様について議論・提案・投票など大きな役割を果たした、www-vrmlメーリングリスト関連。
  • SonyのVirtual Societyページ
    SGIと共にVRML2.0の仕様を提案したソニーのVRMLサイト。VRML2.0ブラウザCommunity Placeなど。
  • Liquid Realityのページ
    Dimension X社のJava言語によるVRML2.0対応本格ブラウザLiquid Realityのページ。
  • VRwaveのページ
    Java言語で作成のVRML2.0ブラウザVRWaveのページ。VRML1.0ブラウザVRWeb、VRMLパーサpwなども。0.9へバージョンアップし、インタラクティブな動きもできるようになったが、まだElevationGridノードをサポートしていない。
  • WorldView
    Intervista社の VRML2.0 ブラウザ。Macintosh 対応プラグイン。
  • CASUS Presenterのページ
    Javaで書かれた、VRML2.0ブラウザCASUS Presenter。OpenInventorのライブラリがなかったので、うちのSunの環境では動かなかった。 X-<
  • FreeWRL
    Perl5で書かれているVRMLブラウザらしい。まだインストールに成功していないので、未確認。
  • http://hiwaay.net/~crispen/vrml/
    いろいろ。ここからもってきたdem2wrl.cには、お世話になりました。
  • EAI (External Authoring Interface)
    • EAI Specification
    • EAI FAQ
    EAI 関連。まだあまり理解してない。^_^;;

オリエンテーリング関係

  • Navigate!
    オーストラリアのLawford氏作成のWindows用OLゲーム。すべての発想の元はここから。
  • Win OL
    同じく、Windows用OLゲーム。インタフェース面などに見るべきものは多い。
  • SRLM
    こちらは、DOS用ゲーム。
  • Mapgen
    地形の自動作成ツール。上記のSRLMは、これをベースにしている。
  • Orienteering Software
    上記のオリエンテーリングゲームにリンクしていたページ。その他にも情報多し。

Java関連

  • SunのJava公式サイト（日本語版）
    JDKの配布やJava API Referenceなど、公式情報がある。英語版は、http://www.java.sun.com/。
  • Java House Mailing List
    Java言語に関するメーリングリストJavaHouse MLのホームページ。トピックスのページは必見。
  • Gamelan
    Java Appletを中心としたJava Class Libraryへのリンク集。

地図関連

  • Digital Map Page
    地形シミューレーションの話題を中心として、地形データなどについて詳細に解説してあるページ。
  • 3D 風景 CG
    3D CG による地形レンダリングソフト・ Kashmir の作者 杉本氏のページ。

    アメリカの数値地図データを管理しているUSGSのデータを提供している。

    実際の数値地図データ (DEM : Data Elevation Model) の解説。
  • Geographic Starting Page
  • 国土地理院
    いわずと知れた、日本の地図を管理・提供している本家・本元。
