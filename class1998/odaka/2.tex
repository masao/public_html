\vspace{-1cm}
{\noindent\bf\Large {\LARGE 2}　　視覚システム}
\vspace{1cm}

この章では、光の物理学、眼の解剖学的・生理学的特性、
視覚システムの機能・機構を扱う。

光は眼に入り、眼の様々な部品を通過して、最終的に網膜に達する。
光がどのように眼に入り、神経細胞での電気的反応に変換されるかについて述べる。
まず光の特性と、その測定法について述べ、
その後、
眼の部品、特に光に焦点を合わせる部品---角膜・レンズ---と
神経システムの信号への変換器---網膜---について述べる。
最後に、視覚情報がどのように解釈されるかについて述べる。

 \section{光}

光は基本的に周波数と強度によって記述できる。
周波数は色（色相）を決定し、強度は明るさを決定する要因となる。

光波は電磁波スペクトルの一つである。
光の周波数は非常に大きいため、波長によって記述される。
電磁波スペクトルの可視部分の波長は約400ナノメートル
\footnote{ナノメートルは$1 × 10^{-9}$メートル}
から約700ナノメートルまでの範囲である。

光の強度は幅広い範囲をとる（最大の光量は最低の光量の約1兆倍）ので、
眼に見える最低の光量との比で記述される。%（デシベル\footnote{デシベルはある強度が別の強度より10の何乗倍強いかを表現する。}）。

 \section{視覚経路}

光が眼に刺激を与えた時に視覚システムが反応する経路について述べる。
角膜、前房水、瞳孔の順に通過していく（図2-1参照）。

  \subsection{瞳孔}

瞳孔は光量の制限を行なう。
明るいと瞳孔が小さくなり、暗いと大きくなる。

\begin{enumerate}
 \item 強度の変化への対応：
光の強度は最大・最低の間で約1兆倍あるが、
瞳孔の直径は最大7〜8mmで最小は2〜3mm位である。
光量の調整はせいぜい16分の1位である。

\item 瞳孔の大きさの調整速度：
大きさの調整は迅速に行なわれない。
うす暗い光から明るい光に変化した時、十分に縮瞳するには約5秒かかる。
明るい光から暗い光に変化した時、完全に散瞳するまでに5分かかる。
\end{enumerate}
以上2点の理由から、瞳孔は光の強度を統制しているとはいえない。
瞳孔の主要な機能はレンズの中心部に光が入るようにすることかもしれない。
また、光強度以外には情動的な要因にも反応するとされており、
一般的に散瞳は情動が深く関わっている。

  \subsection{レンズ}

レンズの主要な機能は物体からくる光に焦点を合わせることである。
筋肉がくっついた膜でレンズを包み、
レンズの厚みを増したり減らしたりすることで焦点を合わせる。

レンズのゆがみ：
\begin{itemize}
 \item 球面収差：レンズの端を通過する光は中心を通過する光と
       おなじ平面上に焦点を結ばないことでおきるゆがみ。
 \item 色収差：光の色が違うことでおきるゆがみ。
\end{itemize}

%レンズ内の細胞は生きているので常に栄養補給が必要だが、
%レンズの構造上血管を通すわけにはいかないので、
%周囲を取り巻く溶液に頼っているが、
%レンズの内部層は十分には栄養補給をしにくいので、
%年齢と共に焦点合わせの機能に障害が起きる。

{\noindent\bf  焦点合わせと輻輳}

{\bf 実験１：}

人間が焦点合わせを行なう際には2つの出来事が起こっている。

\begin{itemize}
\item 輻輳：一組の眼筋が物体に両眼を向けるために両眼の向きを調整する。
\item 焦点合わせ：もう一組の筋肉がレンズの厚みを調整し焦点が合うようにしている。
\end{itemize}

眼が焦点を維持している方法は分かっていない。
焦点を意識的にぼかすこともできるので、
焦点合わせのシステムが脳の高次中枢から独立していないことは確実である。

  \subsection{網膜}

光はレンズと眼の内部を満たしているゼリー状の物質（ガラス体）を通過した後、
最終的に網膜に到達する。

網膜は光のイメージを生理学的な応答に変換する。

光の受容の障害：

網膜は眼球の後部の内側に広がっているが、
光を感じる要素は外側ではなく内側（脳の方）を向いている。
そのため、光の受容器に結合している神経繊維は
入ってくる光の通過する経路上にあり、蜘蛛の巣のように絡み合っている。

さらにこれらの神経細胞は眼の内部にあるので眼から出ていくためには
後部表面のどこかに開口部がなければならない
（これが盲点が存在する原因）。

 \section{光に対する化学反応}

網膜は露光されていない場合、赤みがかった紫色をしており、
視紅という化学物質がある。
光に反応すると、赤みがかった紫→青みがかった黄色→透明（漂白）と
いうように変化する。

夜行性の鳥やけもの類は比較的高濃度の視紅を持っている傾向がある。
また他の爬虫類や昼行性の鳥類・魚は視紅とは異なるが、
光が当たると漂白される物質を持っている。

%波長の違う光でどの程度の漂白が起きるかについての測定値を吸収スペクトルと呼ぶ。

  \subsection{光化学反応}

%眼の中で起きる光化学反応は正確に判明している。
光が存在しない場合、高濃度のロドプシン（視紅の化学名）が存在する。
光が当たると連続的な変化が起こり、レチニンという物質ができる。
この物質は最終的にはビタミンAに還元される。
レチニンは部分的に漂白された物質を黄色に変える。
長く光を受けてビタミンAが優勢になると、
網膜は完全に漂白されて色がなくなる。

%これは2つの化学反応による。
%
%\begin{enumerate}
% \item 
%\end{enumerate}

%化学反応は視覚の感度を決定しているが、
%通常の照明の下では使えるロドプシンのほんのわずかなパーセンテージだけしか
%実際に漂白されない。
%

光化学反応と神経繊維の電気的インパルスの発射との関係は分かっていない。
化学反応がどのように脳への実際の視覚信号を発生するかは未解決である。

 \section{視覚の神経解剖学}

網膜は一様ではなく、莫大な個数の光の受容器からなっている。
それぞれの受容器はそれに当たった光に対して反応する。
光の受容器には杆体(rod)と錘体(cone)の2種類がある。

この2つは全く異なる視覚システムである。

視紅（ロドプシン）は杆体の先端にしか存在しない。
錘体は色覚に必要な光化学物質を含んでいる。

ヒトではそれぞれの眼には約600万個の錘体、約1億2000万個の杆体、
総計約1億2500万個の受容器がある。

網膜上の受容器の分布状態は杆体と錘体とで異なる。
杆体は周辺部で、錘体は中心部で優勢となっている。

眼の中心には中心窩と呼ばれる領域があり、
この領域には高密度の受容器がある。
中心窩は最適な位置にあり、
眼が凝視している点付近のイメージを中心部分で受容する。
また、中心窩は最大視力を持つ場所でもある。

眼は両眼合わせて約2億5000万個の受容器によって視覚情報を集め、
その情報を160万本の神経繊維で脳に送っている。

錘体と杆体の2種の視覚システムはお互いの働きを補完しあっている。
錘体は色の情報を送ることが可能だが、感度には限界がある。
杆体は光に体する感度は高いが、解像力に限界があり、色を感じない。
両システムは一緒に働くことで光信号に柔軟な対応ができる強力なシステムである。

  \subsection{網膜のネットワーク}

眼自体で起こる情報処理：

情報を処理・伝達する神経細胞はニューロンと呼ばれている。
ニューロンは細胞体と細胞相互を接続する軸索（神経繊維）からなっている。
ニューロン間の連結は細胞体か樹状突起のどちらかで起こる。
この連結部をシナプスと呼ぶ。

網膜内ではこれらの細胞が網膜の水平機構を形成している。
網膜はニューロンのネットワークとみなすことができるだろう。

細胞の中には網膜で生じた信号を脳に運ぶ役割をしているものもある。
これらの細胞が網膜の垂直機構を形成している。
垂直機構の最後の細胞は神経節細胞と呼ばれていて、
ほぼ80万個存在する。
この細胞の軸索は10数センチメートルの長さがあり、眼から脳に伸びている。

色々なタイプの神経細胞が杆体と錘体から脳にいたるまでの信号伝達に関わっている。
受容細胞からの信号は2個のシナプスを経由して垂直方向に進行する。
最初のシナプスは受容細胞そのものと両極細胞との間、
第2のシナプスは両極細胞と神経節細胞との間にある。
神経節細胞の軸索は眼から出ていく視神経を構成している。

信号の分析は水平機構でも行なわれ、水平細胞がこれを行なう。
% この水平細胞は受容器そのものを連結していて、
次の高次なレベルでの水平処理はアマクリン細胞が行なう。

相互結合の密度は場所によってことなる。
網膜周辺部では1個の神経節細胞は数千個の杆体から情報をうけとっている。
網膜の中心窩ではそれぞれの錘体が1個の両極細胞を結合して、
1個の神経節細胞に結合している。
この種の直接結合は小型の細胞である、
超小型両極細胞と超小型神経節細胞によって行なわれている。

 \section{脳までの経路}

網膜の神経節細胞はそれぞれの繊維が集まってし神経を作っている。
神経繊維の総数は約80万本でおよそ鉛筆の太さくらいのケーブルである。
それぞれの眼から出た神経繊維は視交叉と呼ばれる場所で交差する。

網膜の左半分から出た神経繊維はすべて脳の右半球に行く。
視交叉で神経繊維は再編成され、視覚場面を2つずつに分ける。
中心窩の右半分に写る視覚場面は脳の右半球に送られる。
凝視点の右側であれば、脳の左半球に送られる。

 \section{視覚情報の抽出}

視覚イメージの処理は特に中心窩にイメージを結ぶ部分に集中している。
錘体の大部分は中心窩に集まっていて、
視覚皮質ではニューロンの50\%以上が視野の10\%にあたる
中心部の解析にあたっているとされている。

眼の周辺部は中心に比べて光への感度は強いが、
中心窩のように視覚場面の細部や色の情報に対しては反応しない。

このような視覚システムは視野のわずかな部分だけが分析されるので、
パターン認知システムを視野の特定部位に向ける方法を持たなければならない。

  \subsection{眼球運動}

眼は眼球運動を行なうことで視覚場面の解析を行なっている。
眼球運動のパターンは視覚場面の不連続な跳躍運動、サッケードからなる。
この運動は1秒間に4〜5回起こり、眼の位置は突発的に高速で移動する。

人間の眼では1秒につき4〜5回以上サッケードが起こることはない。
この運動は弾道的であり、運動は途中での修正ができない。

眼球位置の測定や制御の実際の神経機構はまだ明らかではない。
サッケードは前頭眼球野と結び付いているようである。
この部位に電気刺激が与えられるとサッケードが誘発され、
運動の大きさと方向は挿入された電気刺激の部位に依存する。
左半球を刺激すると右方向のサッケードが、
左半球を刺激すると左方向のサッケードが起こる。

  \subsection{定位チャンネル}

視覚イメージから標的の位置を計算する過程は、
通常のパターン認知に含まれる神経チャンネルとは
別のチャンネルで行なわれているのではないかといわれている。
つまり、視覚イメージの中の物体を識別する際に、
視野内で物体の正確な向きや位置は重要ではない。
しかし、物体の定位に必要なデータは位置と向きである。

この2種類の分析には2つの独立した神経チャンネルが使われているらしい。
眼から脳へと向かう1組の神経繊維は外側膝状核へ向かうものと分かれて、
上丘と呼ばれている脳の中枢に伸びている。
上丘を出た神経繊維は眼球運動や頭の向き、
姿勢の調節などを扱う運動制御システムと結合しているようである。
また、上丘は視覚情報を受容する皮質からも情報を受け取っているらしい。

この上丘が環境中で対象を定位することに関連すると思われる有力な候補である。

このチャンネルのニューロンが電極により活性化すると、
ある種の頭部運動や姿勢の調節が起きる。
さらに視野の特定部分と関連したニューロンが刺激を受けると、
その方向に向きを変える。

上丘の細胞は主に標的を動かすことで活性化される。
視覚・聴覚・触覚から送られた情報は物体を定位するのに役立っている。
また、身体の向きを制御するのにも利用されているようである。

以上の2つの主要な視覚チャンネル（パターン認知・定位）の内の一方が壊れたら、
どんなことが現実に起こるのだろうか。

  \subsection{視覚皮質切除下での視覚}

パターン認知に使われる情報の神経チャンネルは視覚皮質を外科的に切除することで
破壊される。
% ネコとサルでは切除により完全な盲目状態になるようだ。
視覚皮質を切除された動物は環境の中の物体を見たり、
正常な視覚--運動の強調活動をしたり、
視覚信号の判別を必要とする課題を学習できなくなる。
しかし、
視覚皮質を切除しても、明るさと、大きさに基づく判別を学習できる。
また、視覚皮質の無いサルでも対象の定位はある程度うまくいく。
この場合には、対象の動きによって判別しているらしい。

  \subsection{上丘切除下での視覚}

実際に定位情報の分析が、
上丘チャンネルで視覚チャンネルと独立に実行されているかの証明は、
上丘チャンネルを破壊して動物の定位能力を調査すれば良いことになる。

この実験にはハムスターを用いる。
上丘の構造を破壊するとハムスターはあらゆる定位能力を失う。
（図2-21）

以上の実験から定位システムと認知システムの2つの別のシステムが
働いているのではないかということを示している。
