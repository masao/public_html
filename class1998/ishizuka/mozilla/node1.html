Mozillaとは

一般には、Netscape Communications社のWWWブラウザ、 Netscape Navigatorを指す。 同社はNetscape Communicatorの フリーソフトウェア化とソースコードの公開を1998年1月に発表した。 ソースコードは3月に公開されて 世界中の開発者たちの手で現在も開発が進められている、 このNetscape Communicator 5.0の開発バージョンを ここではMozilla（モジラ）と呼ぶ。 Mozillaの公開と開発、それに関連する文書はhttp://www.mozilla.org/で公開されている。

Mozillaのソースコードからのコンパイルは

  • 機種：Sun Ultra-Enterprise 3000
  • OS：Solaris 2.6
  • Cコンパイラ：GNU C ( C++ ) Compiler 2.8.1
という環境で行なった。

現在の配布バージョンは1998年9月4日版で、 mozilla-19980904.tar.gz という名前で配布されており、このファイルの容量は約21.3Mバイトである。 この圧縮された配布ファイルを解凍すると、全体で約159.8Mバイトの容量となる。 （ちなみに、プログラムのソースファイルの数だけでも10000以上ある。）

コンパイルに成功すると、図1のようなウィンドウのMozillaが起動できる。
