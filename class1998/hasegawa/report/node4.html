データ形式

Matrix Marketで提供されている行列のデータファイルの形式には次の3種類がある。

1.
Matrix Market Exchange Formats[3]（資料1参照）

Matrix Marketで利用するために開発されたデータ形式で、 行列データのやりとりするためのテキスト形式のファイルフォーマットである。 表現が簡単にでき、解析しやすいように設計されている。

Matrix Market Exchange Formatsには次の2種類の表現法がある。

(a)
Coordinate Format：疎行列の表現に用いる。
(b)
Array Format：密行列の表現に用いる。

2.
Harwell-Boeing Exchange Format（資料2参照）

疎行列の表現用の形式で固定長フィールドを持つテキスト形式のファイルフォーマットである。 特に、Fortranでの扱いに適した形式になっている。

3.
Coordinate Text File Format （廃止予定）

さらに、これらのデータファイルはgzip形式で圧縮されて提供される。
